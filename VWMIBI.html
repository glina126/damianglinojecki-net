Overview of the Project



You do not own a device, unless you know what drives it.


The idea of hacking cars in the recent days, has been getting a lot of traction from hackers/makers and professionals alike. Dead fish go with the flow, but if it drives them -- that is exactly what they should do.

In this project, I will be attempting at reverse engineering and hacking the MIB I platform from a 2015 VW GTI.

Responsibility Bluetooth Communications

This bluetooth module is rather big, but it does its job well. It has limited range, but as long as you are within the car, it performs like it should. Though... There are many modules out there today that could be a much better fit. VW.. why?

lc4064v5tn-75



Part Number lc4064v5tn-75
Datasheet Link link
Responsibility Unknown - reprogrammable logic gate array

This is one of the most intriguing parts in this system. It is a reprogrammable gate array. In other words, you can create any logical circuit you want (within the gate limit). This allows the user to implement custom "hardware" and on top of that it can be "reprogrammed" on the fly. This could allow for implementation of a specific algorithm.
